# QR And barcode scanner

<h3>Installation</h3><br>
$ sudo apt-get install python3-tk<br>
$ sudo apt-get install libzbar0<br>
$ pip3 install -r requirements.txt


<h3>Usage:</h3><br>
$ python3 qr_read.py<br>
